## 🖋️ ② 各セクションの書き方ルール

### ■ タイトル（SEO）
```
形式：「店名｜エリア｜ジャンル｜アメフト関係者にもおすすめの〇〇 | FOOTBALL CONNECTION」

例：
・東海亭｜西麻布｜焼肉｜アメフト関係者にもおすすめの有名焼肉店
・COTTON FIELDS｜芦屋｜バー｜アメフト関係者にもおすすめの隠れ家バー
・焼肉きゅうこん｜目黒｜焼肉｜アメフト関係者にもおすすめの焼肉店
```

### ■ メタディスクリプション（120〜140文字）
```
必須要素：
① エリア名
② 店の特徴（ジャンル・代表メニュー・雰囲気）
③ 利用シーン（会食・接待・記念日など）
④ アメフト要素（〇〇大学OBが経営 など）

例：
「西麻布交差点すぐの松阪牛炭火焼肉の名店・東海亭。
 立教大学RushersOBのオーナーが営む、西麻布 会食・接待に最適の一軒。
 格のある食事体験をアメフト関係者にも強くおすすめします。」（138文字）
```

### ■ 一言フック（最重要）
```
条件：
・20文字以内
・「行きたい」「知りたい」を引き出す
・断定口調（「〜がある」「〜といえばここ」）

良い例：
✅ 「西麻布で焼肉といえば名前が挙がる一店」
✅ 「人生の節目に選びたくなるレストラン」
✅ 「仲間が自然と集まる、大人の隠れ家」
✅ 「知っているとカッコいい、目黒の焼肉店」

NG例：
❌ 「おいしい焼肉がたべられるお店です」
❌ 「アメフトOBが経営する飲食店」
❌ 「ぜひ一度来てみてください」
```

### ■ リード文（150〜200文字）
```
構成：
1文目：立地・シーンの描写（引き込み）
2文目：オーナーの背景＋開業の文脈
3文目：どんな時間を過ごせるかの余韻

例：
「目黒駅から徒歩すぐ、テルレビルの地下1階へ続く階段を下りると、
 炭火の香りが迎えてくれる。
 法政大学シルバースターOBの阿部洋介オーナーが、
 九州の生産者とのご縁を胸に2022年に開いた一軒。
 仲間と囲む炎と肉の夜は、いつの間にか特別な記憶になる。」
```

### ■ 店舗紹介（強みカード3枚）
```
カード1：料理・サービスの質（具体的な素材・調理法・名物）
カード2：立地・アクセスの価値
カード3：初めてでも安心できる理由

各カード：タイトル＋2〜3文の説明。情緒＋具体性。
```

### ■ ストーリーブロック（OWNER'S STORY）
```
最重要セクション。「なぜこの店が存在するのか」を語る。

構成：
① きっかけ・背景（どんな経験が開業につながったか）
② 想いの核心（何を大切にしているか）→ 太字にする
③ 展望・姿勢（どんな店であり続けたいか）

オーナーの言葉として引用スタイルで書く（「〜」体）。
```

### ■ グレードバナー
```
ジャンル別タイトル：
・焼肉・和食・高級店 → ✦ SETTAI GRADE ✦
・フレンチ・記念日   → ✦ MILESTONE GRADE ✦
・バー・コミュニティ  → ✦ SOCIAL GRADE ✦
・カフェ・日常       → ✦ DAILY GRADE ✦

本文：接待・会食・記念日等の場として格式があることを1〜2文で。
```

### ■ アメフト視点
```
必須要素：
① 大学名・チーム名（正式名称で）
② フットボールで培った価値観と店づくりの関係
③ 仲間と利用する価値・おすすめシーン
④ 締め：「〇〇でいい△△ない？と聞かれたら迷わずここを答えてほしい」
```

## 🏷️ ⑤ SEOキーワード配置チェックリスト

各ページで自然に含めるべきキーワード：

- [ ] `エリア名＋ジャンル`（例：西麻布 焼肉）
- [ ] `エリア名＋会食`（例：西麻布 会食）
- [ ] `エリア名＋おすすめ`（例：西麻布 おすすめ）
- [ ] `エリア名＋接待`（例：西麻布 接待）
- [ ] `大学名＋OB`（例：立教大学 OB）
- [ ] `アメフト 関係者 店`
- [ ] 店名（metaタイトル・H1・本文に最低3回）
